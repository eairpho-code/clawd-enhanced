<h1 align="center">Clawd Enhanced</h1>

<p align="center">
  A community-enhanced fork of Clawd on Desk with AI, memory, and interactive desktop pet system.
</p >

---

## 🚀 Features

### 💬 Interactive Bubble System
- Click the desktop pet to trigger random speech bubbles
- Some bubbles include interactive questions
- User choices affect pet responses dynamically

### 🤖 AI Integration (Optional)
- Supports OpenAI / DeepSeek / Anthropic APIs
- Smart fallback to local mode
- Daily usage limit (50 requests/day) to control cost

### 🌙 Context-Aware Behavior
- Reacts to:
  - Dragging
  - Network status
  - Night-time usage
- More lifelike personality behavior

### 🧠 Memory System
- Remembers recent user interactions
- Adapts responses based on previous choices

### 😴 Sleep Mode
- Pet becomes drowsy and less active
- Personality shifts based on state

### 💬 Chat System
- Built-in chat with 2-turn memory
- Supports continuous lightweight conversation

### 🔑 API Key Panel
- Supports OpenAI / DeepSeek / Anthropic
- Test / Save / Update API keys
- Prevents invalid configuration usage
